Talkthrough アプリケーション READ ME
2005.Jun.10 酔漢

TOPPERS/JSP 1.4.1 for Blackfin ベースの Talkthrough アプリケーションです。全体は3つの部分から構成されています。

- config\blackfin-vdsp\bf533\libkernel.dpj
- _libs\ad1836a.dpj
- _applications\talkthrough.dpj

これらのプロジェクトは_applications\talkthrough\talkthrough.dpg に含まれて居ます。

ビルド環境
- VisualDSP++ 4.0 2005 March Update
- EZ-KIT Lite BF533 Rev 1.5, Chip Rev 0.2

詳細は http://adsp2191.hp.infoseek.co.jp/ まで。
